# Getting Started — Disaster Relief Resource Agent

Requirements: Python 3.11+, uv, Git, Antigravity IDE, Gemini API key.

Workspace:
- agent_builder_playbook_custom.md
- GETTING_STARTED_custom.md

Open the workspace and start with:
@agent_builder_playbook_custom.md start

MVP Architecture:
User -> Emergency Classifier -> Shelter/Medical -> Escalation -> Response

Future (not now): Weather API, Maps, RAG, Multilingual.
