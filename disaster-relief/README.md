# 🌊 Kerala Disaster Relief Portal

An AI-powered multi-agent disaster relief dispatcher built with **Google ADK (Agent Development Kit)** and **Gradio**. The system intelligently classifies emergency requests and routes them to specialized agents — for shelter lookup, medical assistance, critical emergency escalation, general onboarding, or recovery guidelines retrieval — using real-time streaming.
